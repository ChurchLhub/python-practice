# Calculator
A simple command-line calculator built in Python that performs basic arithmetic operations.

## Description
I built this while learning python to practice creating functions and calling them in the program body

## Features
it features all the basic functions a calculator needs.
1. Addition
2. Subtraction
3. Multiplication
4. Division

## Technologies Used
- Python 3
- Git

## How to Run
```bash 
python calculator.py

```
